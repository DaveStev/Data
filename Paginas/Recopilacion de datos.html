Recopilacion de datos

La recopilación de datos, es recolectar información para responder preguntas o resolver problemas. Es como cuando los detectives recogen pistas para resolver un misterio. Puedes recopilar datos de diferentes maneras: haciendo preguntas, observando lo que sucede o mirando documentos.

Por ejemplo, si quieres saber si a tus amigos les gusta más el fútbol o el baloncesto, puedes preguntarles a todos y anotar las respuestas. Luego, puedes contar cuántos amigos prefieren cada deporte y así tendrás datos que te ayudarán a responder tu pregunta. ¡Recopilar datos puede ser muy divertido y útil para aprender cosas nuevas!
